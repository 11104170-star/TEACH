# 國小時間與角度教學

這是一個獨立的 Streamlit 教學網站，內容包含國小數學常見的時間與角度概念。

## 功能

- 互動時鐘：練習讀小時與分鐘
- 經過時間：輸入開始時間，自動計算結束時間
- 互動量角器：認識銳角、直角、鈍角、平角
- 角度加減：練習兩個角合併後的角度
- 小測驗：三題即時檢查答案

## 本機執行

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Cloud 設定

部署時請使用：

```text

```
